Google Sheets 연동 설정
1
Google Sheets 만들기
새 구글 스프레드시트를 만들고, apps-script.js 파일의 코드를 확장 프로그램 → Apps Script에 붙여넣으세요.
그 다음 setupSheets() 함수를 한 번 실행하면 시트 구조가 자동으로 만들어져요.
2
Apps Script 배포
Apps Script 편집기에서 배포 → 새 배포를 클릭하세요.
유형: 웹 앱, 다음 사용자로 실행: 나, 액세스 권한: 모든 사용자로 설정 후 배포하세요.
배포 후 표시되는 웹 앱 URL을 복사하세요.
3
API URL 입력
배포된 웹 앱 URL을 아래에 입력하세요.
4
팀원 정보 입력 (시트에서)
구글 시트의 멤버 탭에 팀원 정보를 입력하세요.
color 값은 HEX 코드 사용 (예: #378ADD)
id | name | color | role
mj | 홍길동 | #378ADD | 팀장
sy | 이서연 | #1D9E75 | 디자이너
5
GitHub Pages 배포 (선택)
이 index.html 파일을 GitHub 저장소에 올리고 Pages를 활성화하면 팀 전체가 URL로 접속할 수 있어요.
저장소 → Settings → Pages → Source: main branch / root
